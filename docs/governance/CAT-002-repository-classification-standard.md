# CAT-002 Repository Classification Standard

Status: Draft
Owner: catylist
Version: 0.1.0

## Abstract

CAT-002 defines the repository classes used by the Catalyst engineering ecosystem.

Repository classification identifies the primary responsibility of each repository and establishes the dependency rules used to keep governance, standards, schemas, automation, templates, execution workflows, and project implementations separated.

## Purpose

Define the authoritative repository classification model for Catalyst repositories.

This standard exists to ensure that each repository has a clear purpose, owner, responsibility, and relationship to the rest of the ecosystem.

## Scope

This standard applies to all repositories participating in the Catalyst engineering ecosystem.

Each repository should declare its repository class in its repository manifest once AES-003 is adopted.
